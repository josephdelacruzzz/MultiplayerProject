**Multiplayer Project**  
Tic-Tac-Toe

How to run:
1. Clone the repository
2. Open UnityHub and select the repository the folder was cloned to
3. Navigate to HelloWorldManager.cs and change "192.168.1.213" to the IP of the host's device.
4. Go to Build -> Build Profiles and configure build for host device and client device
5. Move the client build to the client device
6. Run the host device's build
7. Run the client device's build
8. Click "Player 1" on the host device
9. Click "Player 2" on the client device
10. Enjoy Tic-Tac-Toe!

This project was ran with Apple Silicon and Apple Intel MacBooks.
